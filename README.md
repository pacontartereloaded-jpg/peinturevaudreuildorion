# Peinture Vaudreuil

Site vitrine d'une équipe de peintres spécialisés en peinture avant emménagement à Vaudreuil-Dorion, Île-Perrot, Hudson.

## Stack
HTML5 + CSS3 + JavaScript vanilla. Aucune dépendance.

## Sections signature
- **Timeline déménagement** — 5 jalons (J−14 / J−7 / J−5 / J−1 / Jour J) sur ligne horizontale
- **Checklist déménagement** — 3 cards (avant la visite / avant le chantier / avant votre arrivée) avec cases à cocher
- **Form avec date emménagement** — date de notarié + date du déménageur en champs séparés
- **Services** — 6 services pensés pour nouveaux propriétaires

## Stack visuel
- Palette : vert d'eau #a8c8b9 / crème #fff8ef / charbon #1a1a1a
- Typo : DM Serif Display + Outfit
- Animations : reveals 650ms cubic-bezier, dot pulse vert d'eau, hover lifts subtle

## Déploiement (Vercel)
1. Importer ce dossier comme nouveau projet sur Vercel.
2. Aucune commande de build (site statique).
3. Pointer le domaine `peinturevaudreuildorion.ca` vers Vercel.
